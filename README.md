# Arm System Register Ref

- Arm System Register Ref (ASRF) is a static web page that users can use to search and check the system register of the aarch64
- ASRF has a following input:
  - Text box to select System Register
    - suggets system registers based on input
    - case insensitive
  - value input
    - accepts hex or decimal value
    - decodes the value and shows the indication of the bits
  - reference link to Arm Architecture Reference Manual (ARM ARM) for the system register
